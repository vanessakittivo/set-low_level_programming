# 0x00-hello_world

This directory contains C programming basics and compilation scripts.

## Files
- 0-preprocessor: Script to run C file through preprocessor
- 1-compiler: Script to compile C file without linking
- 2-assembler: Script to generate assembly code
- 3-name: Script to compile and create executable named cisfun
- 4-puts.c: C program using puts
- 5-printf.c: C program using printf
- 6-size.c: C program printing size of types
