# C - Functions, nested loops

This repository contains solutions for the **0x02. C - Functions, nested loops** project.

## Tasks

0. **_putchar** - Program that prints _putchar
1. **I sometimes suffer from insomnia...** - Function that prints the alphabet
2. **10 x alphabet** - Function that prints alphabet 10 times
3. **islower** - Function that checks for lowercase character
4. **isalpha** - Function that checks for alphabetic character
5. **Sign** - Function that prints the sign of a number
6. **There is no such thing as absolute value...** - Function that computes absolute value
7. **There are only 3 colors...** - Function that prints last digit
8. **I'm federal agent Jack Bauer...** - Function that prints every minute of the day
9. **Learn your times table** - Function that prints the 9 times table
10. **a + b** - Function that adds two integers
11. **98 Battery Street, the OG** - Function that prints numbers to 98
