# More Malloc, Free

This directory contains functions for dynamic memory allocation in C.

## Files

| File | Description |
|------|-------------|
| 0-malloc_checked.c | Wrapper for malloc that exits with 98 on failure |
| 1-string_nconcat.c | Concatenates two strings with n bytes from s2 |
| 2-calloc.c | Allocates and zero-initializes memory |
| 3-array_range.c | Creates array of integers from min to max |

## Compilation

All files compiled with:
gcc -Wall -pedantic -Werror -Wextra -std=gnu89
