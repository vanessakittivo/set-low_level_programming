# malloc_free - Dynamic Memory Allocation in C

This directory contains functions for dynamic memory allocation using malloc and free.

## Files

- `0-create_array.c` - Creates an array of chars initialized with a specific char
- `1-strdup.c` - Returns a pointer to a duplicated string
- `2-str_concat.c` - Concatenates two strings
- `3-alloc_grid.c` - Returns a pointer to a 2D array of integers
- `4-free_grid.c` - Frees a 2D grid

## Compilation

All files are compiled with:
gcc -Wall -pedantic -Werror -Wextra -std=gnu89
