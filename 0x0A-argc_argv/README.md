# 0x0A. argc, argv

## Description
This project explores how to handle **command-line arguments** in C using `argc` and `argv`.

## Tasks

- **0. It ain't what they call you, it's what you answer to**  
  [`0-whatsmyname.c`](0-whatsmyname.c) — Prints the name of the program.

- **1. Silence is argument carried out by other means**  
  [`1-args.c`](1-args.c) — Prints the number of arguments passed.

- **2. The best argument against democracy is a five-minute conversation with the average voter**  
  [`2-args.c`](2-args.c) — Prints all arguments received.

- **3. Neither irony nor sarcasm is argument**  
  [`3-mul.c`](3-mul.c) — Multiplies two numbers given as arguments.

- **4. To infinity and beyond**  
  [`4-add.c`](4-add.c) — Adds positive numbers passed as arguments.

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- Compiled on Ubuntu 20.04 LTS using `gcc -Wall -Werror -Wextra -pedantic -std=gnu89`
- All files end with a new line
- Betty style compliant
- No global variables
- Maximum 5 functions per file
- Prototypes in `main.h`

- `main.h` — All prototypes
