# C - Recursion

This repository contains solutions for the **0x08. C - Recursion** project.

## Tasks

0. **She locked away a secret...** - Function that prints a string using recursion
1. **Why is it so important to dream?** - Function that prints a string in reverse using recursion
2. **Dreams feel real while we're in them...** - Function that returns string length using recursion
3. **You mustn't be afraid to dream a little bigger...** - Function that returns factorial using recursion
4. **Once an idea has taken hold of the brain...** - Function that returns x^y using recursion
5. **Your subconscious is looking for the dreamer** - Function that returns natural square root using recursion
6. **Inception. Is it possible?** - Function that checks if a number is prime using recursion
