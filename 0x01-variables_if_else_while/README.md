# C - Variables, if, else, while

This repository contains solutions for the **0x01. C - Variables, if, else, while** project.

## Tasks

0. **Positive anything is better than negative nothing** - Program that prints whether a random number is positive or negative.
1. **The last digit** - Program that prints the last digit of a random number.
2. **I sometimes suffer from insomnia...** - Program that prints the alphabet in lowercase.
3. **alphABET** - Program that prints the alphabet in lowercase then uppercase.
4. **When I was having that alphabet soup...** - Program that prints alphabet except q and e.
5. **Numbers** - Program that prints all single digit numbers of base 10.
6. **Numberz** - Program that prints all single digit numbers without char variables.
7. **Smile in the mirror** - Program that prints alphabet in reverse.
8. **Hexadecimal** - Program that prints all numbers of base 16.
9. **Patience, persistence and perspiration...** - Program that prints combinations of single-digit numbers.
