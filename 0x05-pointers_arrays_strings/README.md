# C - Pointers, arrays and strings

This repository contains solutions for the **0x05. C - Pointers, arrays and strings** project.

## Tasks

0. **98 Battery st.** - Function that updates value to 98 using pointer
1. **Don't swap horses** - Function that swaps values of two integers
2. **String length** - Function that returns length of a string
3. **Print string** - Function that prints a string
4. **Print reverse** - Function that prints a string in reverse
5. **Reverse string** - Function that reverses a string
6. **Every other character** - Function that prints every other character
7. **Half the string** - Function that prints half of a string
8. **Print array** - Function that prints elements of an array
9. **strcpy** - Function that copies a string
