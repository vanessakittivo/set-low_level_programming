# C - More pointers, arrays and strings

This repository contains solutions for the **0x06. C - More pointers, arrays and strings** project.

## Tasks

0. **strcat** - Function that concatenates two strings
1. **strncat** - Function that concatenates two strings using at most n bytes
2. **strncpy** - Function that copies a string
3. **strcmp** - Function that compares two strings
4. **I am a kind of paranoid in reverse** - Function that reverses an array
5. **Always look up** - Function that changes lowercase to uppercase
6. **Expect the best** - Function that capitalizes all words
7. **Mozart composed his music...** - Function that encodes a string into 1337
