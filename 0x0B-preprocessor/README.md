# 0x0B. C - Preprocessor

## Description
This project explores the C preprocessor: macros, include guards, conditional compilation, and function-like macros.

## Tasks

- **0. Object-like Macro**  
  [`0-object_like_macro.h`](0-object_like_macro.h) — Defines `SIZE` as 1024

- **1. Pi**  
  [`1-pi.h`](1-pi.h) — Defines `PI` as 3.14159265359

- **2. File name**  
  [`2-main.c`](2-main.c) — Prints the name of the file it was compiled from using `__FILE__`

- **3. Function-like macro**  
  [`3-function_like_macro.h`](3-function_like_macro.h) — Defines `ABS(x)` macro

- **4. SUM**  
  [`4-sum.h`](4-sum.h) — Defines `SUM(x, y)` macro
